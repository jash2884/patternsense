PatternSense is a mobile app that detects the underlying DSA problem-solving pattern from a given problem statement.

## Features
- Detects common DSA patterns (Sliding Window, DP, Graphs, Two Pointers)
- Clean mobile-first UI
- Keyboard-safe UX for iOS and Android
- Built with React Native & Expo

## Tech Stack
- React Native
- Expo Router
- TypeScript
- JavaScript Logic Engine (rule-based v1)

## Future Enhancements
- Pattern confidence score
- Explanation of why a pattern was chosen
- Similar problem suggestions
- ML-based classification
